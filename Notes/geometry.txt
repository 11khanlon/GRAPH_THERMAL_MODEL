Every python file has a special built-in variable called __name__
Case 1: You run the file directly 
then python sets __name__ = "__main__"
so this code then runs:

if __name__ == "__main__":
    geom = Geometry("example_part.stl")
    geom.build_blocks()
    print(geom.blocks[0])

Case 2: Another file imports it 
Suppose main.py contains 
from geometry import Geometry

geom = Geometry("part.stl")
geom.build_blocks()

when geometry.py is imported Python sets __name__ = "geometry" 
now the condition if __name__ == "__main__":  is skipped
